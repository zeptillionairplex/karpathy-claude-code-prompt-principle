# GSD List Phase Assumptions Workflow

Workflow for documenting and reviewing phase assumptions.

## When to Use

- Reviewing phase dependencies
- Identifying risks
- Planning mitigation

## Phases

1. Load phase details
2. Document assumptions
3. Highlight dependencies
4. Plan risk mitigation

## Success Criteria

Assumptions documented and risks mitigated.
