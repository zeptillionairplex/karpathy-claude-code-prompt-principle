# Style Selection (HIGH)

- `style-match` — Match style to product type and industry
- `consistency` — Use same style across all pages; never mix flat and skeuomorphic randomly
- `no-emoji-icons` — Use SVG icons (Heroicons, Lucide), not emojis
- `color-palette-from-product` — Choose palette from product type and industry context
- `effects-match-style` — Shadows, blur, radius aligned with chosen style (glass / flat / clay)
- `platform-adaptive` — Respect platform idioms: iOS HIG vs Material navigation and controls
- `state-clarity` — Hover/pressed/disabled states visually distinct while staying on-style
- `elevation-consistent` — Consistent shadow scale for cards, sheets, modals
- `dark-mode-pairing` — Design light/dark variants together for consistent brand and contrast
- `icon-style-consistent` — One icon set across the product (same stroke width, corner radius)
- `system-controls` — Prefer native controls; customize only when branding requires it
- `blur-purpose` — Blur signals background dismissal (modals, sheets); not decoration
- `primary-action` — One primary CTA per screen; secondary actions visually subordinate
