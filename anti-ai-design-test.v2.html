세 가지 변화. 모든 작업에서 한계를 넘어.

M5 Max · chip

CPU 16코어, GPU 40코어. 압도적인 성능.

16코어 CPU 와 최대 40코어 GPU. M4 Max 대비 CPU 35%, GPU 50% 빨라졌습니다. 8K 영상 편집과 대규모 LLM 추론을 노트북 한 대로 처리합니다.

디스플레이 · display

Liquid Retina XDR. 모든 픽셀을 정확하게.

ProMotion 120Hz, 최대 1,600니트 HDR. 색역 P3, 표현은 진실 그대로.

배터리 · battery

최대 22시간. 하루를 충분히.

한 번 충전으로 영상 재생 22시간. 이동 중에도 어댑터 없이.

design note · 02

한 장의 알루미늄에서 — 한 대의 도구로.

MacBook Pro 의 케이스는 100% 재활용 알루미늄을 단일 블록에서 깎아 냅니다. 나사 한 개도 정확한 위치에서 정확한 각도로 들어갑니다.

내부 회로는 모두 100% 재활용 구리, 100% 재활용 희토류로 만들어집니다. 탄소 중립 제품 라인의 일부입니다.
